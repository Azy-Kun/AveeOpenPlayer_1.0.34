Yahoo Media RSS supplements the enclosure capabilities of RSS 2.0 to handle additional information and media types. "Media RSS" is a new RSS module that supplements the enclosure capabilities of RSS 2.0 to allow for more robust media syndication. RSS enclosures are already being used to syndicate audio files and images. Media RSS extends enclosures to handle other media types, such as short films or TV, as well as provide additional metadata with the media. Media RSS extension enables content publishers and bloggers to syndicate multimedia content such as TV and video clips, movies, images, and audio. It also allows content to be represented in multiple formats.
